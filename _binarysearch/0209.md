#### Problem statement

#### Solution
Equal to Leetcode 0581. Shortest Unsorted Continuous Subarray.


#### Complexity
It is `O(n log n)` for time and `O(n)` for space here.

#### Code
```python
class Solution:
    def solve(self, nums):
        nums2 = sorted(nums)
        n = len(nums)
        if nums2 == nums: return 0
        arr = [i for i in range(n) if nums[i] != nums2[i]]
        return max(arr) - min(arr) + 1
```
