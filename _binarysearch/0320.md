#### Solution
Variation of Leetcode 0049. Group Anagrams.

#### Complexity
Time complexity will be `O(nk * log k)`, space complexity is `O(nk)`.

#### Code
```python
class Solution:
    def solve(self, words):
        t = defaultdict(list)
        for s in words:
            t["".join(sorted(s))].append(s)
        return max(len(x) for x in t.values())
```
