#### Problem statement

#### Solution
Equal to Leetcode 0453 Minimum Moves to Equal Array Elements.

#### Complexity
It is `O(n)` for time and `O(1)` for space.

#### Code
```python
class Solution:
    def solve(self, nums):
        return sum(nums) - len(nums)*min(nums)
```
