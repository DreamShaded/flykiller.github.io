#### Problem statement

<a href="https://binarysearch.com/problems/High-Frequency/"> <font color = blue>https://binarysearch.com/problems/High-Frequency/

#### Solution
Just use counter.

#### Complexity
It is `O(n)` for time and space.

#### Code
```python
class Solution:
    def solve(self, nums):
        if not nums: return 0
        return max(Counter(nums).values())
```
