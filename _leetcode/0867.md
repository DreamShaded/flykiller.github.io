#### Solution
In python it can be done with one simple line with `O(mn)` complexity. Also we can return `list()` if needed in the end.

#### Complexity
Time and space is `O(mn)`.

#### Code
```python
class Solution:
    def transpose(self, matrix):
        return zip(*matrix)
```
