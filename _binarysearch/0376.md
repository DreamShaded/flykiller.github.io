#### Solution
Actually, problem statement is not correct, there are a lot of Gray codes, but we need to choose specific one. Element can be calculated with direct formula.

#### Complexity
It is `O(1)` for time and space.

#### Code
```python
class Solution:
    def solve(self, n):
        return n ^ (n >> 1)
```
