#### Solution
Equal to leetcode 0115. Distinct Subsequences.

#### Complexity
It i s`O(mn)` for time and space.

#### Code
```python
class Solution:
    def solve(self, s, t):
        @lru_cache(None)
        def dp(i, j):
            if i == -1: return j == -1
            if j == -1: return j == -1
            return (dp(i-1, j) + (s[i] == t[j]) * dp(i-1, j-1)) % (10**9 + 7)
        
        if not t: return 0
        return dp(len(s) - 1, len(t) - 1)
```
